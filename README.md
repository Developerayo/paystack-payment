## Usage

Run locally:

Clone
```
git clone https://github.com/Developerayo/paystack-payment.git
```

Move into the file with
```
cd paystack-payment
```

Then
```
yarn install
```
Start dev server

```
yarn start
```
now open https://localhost:3000.

Testing with jest and standards

```
yarn test
```
